# 🌐 Portfolio Website

A modern and interactive **personal portfolio website** built using **HTML, CSS, and JavaScript** to showcase my skills, projects, and contact information.
The portfolio features smooth animations, an interactive particle background, custom cursor effects, and responsive design to provide an engaging and visually appealing user experience.

This project reflects my interest in **Full Stack Web Development** and highlights my work with technologies such as **React, Node.js, and Python**, along with modern frontend UI techniques.

---

# 🚀 Features

* ✨ Modern and clean UI design
* 🎯 Smooth scrolling navigation
* 🌌 Interactive **particle background** using Particles.js
* 🖱️ Custom glowing cursor effect
* 🪄 Scroll reveal animations using ScrollReveal.js
* 🧊 3D tilt project cards using VanillaTilt.js
* 📊 Animated skill progress bars
* 📱 Responsive layout for different screen sizes
* 📧 Contact form with email integration
* 🔗 Social links for GitHub and LinkedIn

---

# 🛠️ Technologies Used

### Frontend

* HTML5

### Libraries & Tools

* **Particles.js** – Animated background particles
* **ScrollReveal.js** – Scroll animations
* **VanillaTilt.js** – 3D tilt effects for project cards
